# Tic Tac Toe

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

* Two-player gameplay
* Custom player names
* Winner and tie detection
* Restart game button
* Prevents selecting occupied cells

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```text
tic-tac-toe/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser

## Concepts Practiced

* Factory Functions
* Module Pattern (IIFE)
* DOM Manipulation
* Game Logic
* Separation of Concerns
